Table

Tables are a common way to give an html a page structure as well

  • the <table>tag - to create a table
  • the <caption>tag - lists a caption for the table
  • the <thead>tag - for content of header row
  • the <tr>tag - for a new row
  • the <th>tag - for a header cell
  • the <tbody>tag - for the body of a cell
  • the <td>tag - for the content of a cell
  • the <tfoot>tag - for the footer row
